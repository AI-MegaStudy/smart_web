# 예은 파트 작업 공유

작성일: 2026-05-01

## 오늘 작업 범위

- 예은 화면 라우트 정리
  - 상품 목록: `/products`
  - 상품 상세: `/product-detail`
  - 예약함: `/cart`
  - 주문서: `/order-form`
  - 결제: `/payment`
  - 예약 확인: `/reservation-confirm`
  - 이메일 인증: `/email-verification`
- 상품 목록/상세/예약함/예약현황 화면을 디자인 참고 이미지 톤에 맞춰 개선했습니다.
- 누락되어 있던 이메일 인증 페이지와 예약 확인 페이지를 추가했습니다.
- 깨진 한글 문구와 일부 레이아웃 overflow를 수정했습니다.

## 백엔드 담당자 확인 필요

- 상품 데이터 API
  - 상품명, 가격, 이미지 URL, 수확 예정일, 잔여 수량, 예약 가능 여부
- 예약/주문 API
  - 예약함 추가, 수량 변경, 주문서 저장, 결제 완료 처리
- 인증 API
  - 회원가입, 로그인, 이메일 인증번호 발송/검증
- 주문 조회 API
  - 최근 예약 주문, 주문 상세, 배송/수확 진행 상태

## 프론트 담당자 공유

- 현재 예은 파트 코드는 `lib/yeeun` 하위에서 작업했습니다.
- `lib/main.dart`는 예은 라우트 진입을 위해 사용 중입니다.
- 공통 라우트는 `lib/yeeun/core/app_routes.dart`에서 관리합니다.
- 현재 상품 데이터는 `lib/yeeun/vm/product_view_model.dart`의 샘플 데이터입니다.

## 팀장 공유

- 현재 구현은 백엔드 연결 전 UI/흐름 확인용입니다.
- 우선 흐름은 `상품 목록 -> 상품 상세 -> 예약함 -> 주문서 -> 결제 -> 예약 확인` 기준으로 맞췄습니다.
- `flutter analyze`, `flutter test` 통과 상태에서 푸시합니다.
